Once you've found the assistant you want to use

<Steps>
  <Step>
  Click “Add Assistant” on its page
  </Step>
  <Step>
  Add any required inputs (e.g. secrets like API keys)
  </Step>
  <Step>
  Select “Save changes” in assistant sidebar on the right hand side
  </Step>
</Steps>

After this, you can then go to your IDE extension and begin using the assistant by selecting it from the assistant dropdown.
